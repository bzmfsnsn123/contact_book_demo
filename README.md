# Contact Book Demo
## Python控制台简易通讯录管理系统
新手练手项目，实现联系人增删改查+本地文件持久化存储，巩固字典、文件读写、循环、异常捕获等基础语法。

## 项目功能
1. 新增联系人：校验姓名非空、手机号11位数字，重复姓名拦截
2. 全部联系人展示：格式化对齐表格，清晰查看所有数据
3. 模糊搜索：输入关键字匹配包含该文字的联系人
4. 修改手机号：选中已有联系人更新号码
5. 删除联系人：一键移除指定记录
6. 数据持久化：信息保存在contact.txt，重启程序数据不丢失
7. 输入容错：非法数字、错误手机号、空白内容均有提示，程序不崩溃

## 运行方式
1. 源码保存为 contact_book.py
2. 直接运行，同目录自动生成 contact.txt 存储联系人数据
3. 菜单数字选择对应功能，0退出程序

## 练习知识点
1. 字典存储联系人：姓名为键、手机号为值，快速增删改查
2. 文件读写with上下文，分行读取/写入文本
3. for循环遍历字典 items()
4. f-string格式化对齐输出表格
5. while True主菜单循环 + break退出
6. try-except捕获非数字输入异常
7. 字符串判断：isdigit()、关键字模糊匹配
8. 字符串分割 strip()、split() 处理文本数据
